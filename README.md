# 🏦 Enterprise-Grade Bank Management System
### 🛡️ Scalable C++ Architecture | Object-Oriented Design Patterns

---

## 🏗️ Architectural Overview
This is not just a console app; it's a demonstration of **Modular Software Engineering**. The system is built using a **Strict Layered Architecture** to ensure separation of concerns and high maintainability.

* **⚡ Logic Layer (Core):** Implements complex financial algorithms, currency conversions using specialized exchange classes, and robust business rules.
* **🖥️ Presentation Layer (Screens):** A high-fidelity UI management system that handles complex user interactions using static class controllers.
* **💾 Data Access Layer (Files):** Optimized file-stream handling for persistent data storage with ACID-like consistency in text-based databases.
* **🛠️ Utility Framework (Lib):** A custom-built standard library for enterprise validation, date-time manipulation, and cryptographic-ready utility functions.

---

## 🚀 Advanced Technical Features
- **Sophisticated Permission Matrix:** Granular access control system implementing a bitwise-inspired permission logic.
- **Transaction Integrity:** Transfer systems designed with state-validation to prevent data loss or double-entry errors.
- **Currency Intelligence:** A multi-currency core capable of real-time-like rate updates and cross-border calculation logic.
- **Input Sanitization:** Military-grade input validation to ensure the system is immune to common buffer or type-mismatch exploits.
