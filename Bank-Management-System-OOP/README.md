# 🏦 Bank Management System (Pure OOP)
### 🛡️ High-Logic Financial Application | C++

Welcome to the **Bank Management System**, a professional console-based application built with **Object-Oriented Programming (OOP)**. This project demonstrates advanced logic, modular architecture, and secure data handling, developed as part of my advanced programming journey.

---

## 🏗️ System Architecture & Organization
The project is structured into logical layers to ensure clean code and easy scalability:

* **📂 Core/**: Business logic and data models (`ClsBankClient`, `ClsUser`, `ClsCurrency`).
* **📂 Screens/**: UI Logic for all system menus and operations (Login, Transactions, Management).
* **📂 Lib/**: Custom utility and validation libraries (`ClsInputValidate`, `ClsDate`, `clsUtil`).
* **📂 Data/**: Database persistence using text files (`.txt`).
* **📂 ScreenShot/**: Full visual documentation of the system.

---

## 🌟 Key Features
- **Security:** Multi-level permissions for users and secure login system.
- **Transactions:** Deposit, Withdraw, and Transfer logic with real-time logging.
- **Currency Exchange:** Integrated currency converter and exchange rate manager.
- **Data Integrity:** Robust input validation library to prevent system crashes.
- **Logging:** Full audit trail for transactions and login registers.
